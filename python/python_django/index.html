Django

  • 第一步、安装python
  • 第二部、安装在Django
    • pip install Django==2.2.4

一、创建项目

  • django-admin startproject <项目名>
  • django-admin startproject mysite

二、启动本地服务

  • python manage.py runserver <后面可以自定义IP：端口号>
  • 创建admin超级管理员
    1. python manage.py migrate 迁移创建数据库生成db.sqlite3
    2. python manage.py createsuperuser 创建超级用户

三、基本应用结构

  • Django 创建应用

    1. python manage.py startapp <应用名称>

  • 同步数据库

    1. python manage.py makemigrations 制造迁移
    2. python manage.py migrate 迁移

  • 在app文件下的admin.py文件中

    1. 注册到admin中:
      • from .models import <模板类名>
      • admin.site.register(模板类名)
    2. 在models.py中写模板类 
      from django.db import models
# Create your models here.
class Article(models.Model):
    title = models.CharField(max_length=30)# CharField字符
    content = models.TextField()# TextField文本

    def __str__(self):
        return f'标题：{self.title}，文章：{self.content}'

pip一键导出和安装（拓展）

  • pip freeze > requirements.txt 将项目这种所使用到的库名导入到requitements.txt文本中
  • pip install -r requirements.txt 直接从requirements.txt取出库名安装到项目中
