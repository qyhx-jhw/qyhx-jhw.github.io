Web 自动化测试

  1. 让程序代替人为自动化验证Web项目功能的过程

为什么Web项目是和自动化测试？

  1. 需求变动不频繁
  2. 新项目周期长
  3. 项目需要回归测试

主流测试工具

  1. QTP（收费）
    • QTP时商业的功能测试工具，收费，支持web，桌面自动化测试
  2. Selenium（开源）
    • Selenium是开源的web自动化测试工具，免费，主要做功能测试
  3. Jmeter（开源，Web，接口，性能）
    • Jmeter是由Apache公司使用Java平台开发的一款调试工具，指出（web，接口测试，型嫩测试）
    • Web测试在通信层（无UI界面）
  4. Loadrunner（收费，Web，性能）
    • loadrunner是商业性 能测试工具，收费，功能强大，适合做复杂场景性能测试。

Selenium特点

  1. 开源
  2. 跨平台：Linux，Windows，mac
  3. 核心功能，可以在多个浏览器上进行自动化测试
  4. 多语言，Java，python，C# JavaScript，rubsy
  5. 成熟稳定，目前被google，百度，腾讯广泛使用

python使用Selenium

  • 安装Selenium 
        pip install selenium
  • 安装Google浏览器的chromeddriver
    • https://www.seleniumhq.org/download/
    • https://sites.google.com/a/chromium.org/chromedriver/
    • 下载对用版本的chromdriver
    • 解压赋值粘贴到google安装目录/或者python的安装目录下
  • 使用 
        from selenium import webdriver #导入
    import time
    driver = webdriver.Chrome()# 创建driver类
    driver.get('https://www.baidu.com')# url百度
    time.sleep(2)

    driver.quit()# 退出driver

WebDriver元素定位

  1. id
  2. name
  3. class_name
  4. tag_name
  5. link_text
  6. partial_link_text
  7. Xpath
  8. Css

  • id定位

    from selenium import webdriver

driver = webdriver.Chrome()
url = 'https://www.baidu.com'
driver.get(url)

input = driver.find_element_by_id('kw') #id进行定位
input.send_keys('python') # 输入框输入'python'字符串

  • name

        input = driver.find_element_by_name('wd')

  • class_name

        input = driver.find_element_class_name('s_ipt')

  • tag_name 拿到所有’input’标签的对象，生成一个所有输入框对像组成的列表

        inputs = driver.find_elements_by_tag_name('input')
    print(inputs)
    print(type(inputs))
    for i in inputs:
        print(i, type(i))

  • link_text

        from selenium import webdriver
    driver = webdriver.Chrome()
    blbl = 'https://search.bilibili.com/'
    driver.get(blbl)
    input = driver.find_element_by_link_text('全站排行')
    input.click()

  • Xpath

    1. XPath即未XML Path的简称，它是一种用来稳定XML/HTML文档中某部分位置的语言
    2. HTML可以开做是XML的一种实现
    3. Xpath定位：
        input1 = driver.find_element_by_xpath("//*[@id='kw']")
    input1.send_keys('python')

  • CSS定位

    1. id选择器
      .class .intro 选择class='intro’的所有元素

      olement p 选择所有<p>元素

    2. class选择器

    • 格式.class 如：.telA <选择class属性值未telA的所有对象>
    1. 元素选择器：input
    2. 属性选择器
    3. 层级选择器

鼠标操作

  • 鼠标点击 
        from selenium import webdriver
    from selenium.webdriver.common.action_chains import ActionChains #导入鼠标模块
    driver = webdriver.Chrome()#初始化网络驱动
    action = ActionChains(driver)#初始化鼠标将driver传入鼠标
    url = 'https://www.baidu.com'
    driver.get(url) # 访问百度
    input1 = driver.find_element_by_id('kw') #拿到输入框
    input1.send_keys('python') # 输入‘python’字符串
    action.double_click(input1).perform()# 双击输入框
  • 鼠标拖动 
    
  img = driver.find_element_by_class_name('main_img.img-hover')#获得图片
  inp = driver.find_element_by_id('kw')#获得位置
  action.drag_and_drop(img, inp).perform() #调用函数将img拖到inp
