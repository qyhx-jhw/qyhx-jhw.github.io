创建文件

  • touch命令创建文件

    [root@CentOS7 ~/dir1]#touch test_one
[root@CentOS7 ~/dir1]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 15:41 test_one

复制文件

  • cp命令完成复制文件的操作：cp source[要复制的文件] destination[目标]
  • 复制于test_one一样的文件test_tow
  [root@CentOS7 ~/dir1]#cp test_one test_tow
  [root@CentOS7 ~/dir1]#ll
  total 0
  -rw-r--r--. 1 root root 0 Jun 27 15:41 test_one
  -rw-r--r--. 1 root root 0 Jun 27 15:47 test_tow
  [root@CentOS7 ~/dir1]#
  • cp -i 新的文件已经存在（test_tow），强制询问是否需要覆盖原文件
  • 也可以复制到现有的其它目录中cp -i test_one ../dir2 
[root@CentOS7 ~/dir2]#cd ../dir1/
[root@CentOS7 ~/dir1]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 15:41 test_one
-rw-r--r--. 1 root root 0 Jun 27 15:51 test_tow
[root@CentOS7 ~/dir1]#cp -i test_one ../dir2

[root@CentOS7 ~/dir1]#cd ../dir2
[root@CentOS7 ~/dir2]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 15:52 test_one
  • cp -R 可以递归地复制整个目录的内容。
[root@CentOS7 ~]#cp -R dir1/ dir3
[root@CentOS7 ~]#cd dir3/
[root@CentOS7 ~/dir3]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 16:01 test_one
-rw-r--r--. 1 root root 0 Jun 27 16:01 test_tow

重命名/移动文件

  • mv命令可以将文件和目录移动到另一个位置或重命名
    • 将test_one 重命名为test_1 
      [root@CentOS7 ~/dir1]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 15:41 test_one
-rw-r--r--. 1 root root 0 Jun 27 15:51 test_tow
[root@CentOS7 ~/dir1]#mv test_one test_1
[root@CentOS7 ~/dir1]#ll
total 0
-rw-r--r--. 1 root root 0 Jun 27 15:41 test_1
-rw-r--r--. 1 root root 0 Jun 27 15:51 test_tow
    • mv移文件位置 
      [root@CentOS7 ~/dir1]#ll
    total 0
    -rw-r--r--. 1 root root 0 Jun 27 15:41 test_1
    -rw-r--r--. 1 root root 0 Jun 27 15:51 test_tow
    [root@CentOS7 ~/dir1]#mv test_1 ../dir2/
    [root@CentOS7 ~/dir1]#ll
    total 0
    -rw-r--r--. 1 root root 0 Jun 27 15:51 test_tow
    [root@CentOS7 ~/dir1]#cd ../dir2/
    [root@CentOS7 ~/dir2]#ll
    total 0
    -rw-r--r--. 1 root root 0 Jun 27 15:41 test_1
    [root@CentOS7 ~/dir2]#

删除文件

  • 在Linux中删除（deleting）叫做移除（removing）。
  • rm命令是删除命令:
    • -i 命令参数提示你是不是要真的删除该文件
      [root@CentOS7 ~/dir2]#ll
  total 0
  -rw-r--r--. 1 root root 0 Jun 27 15:41 test_1
  [root@CentOS7 ~/dir2]#rm -i test_1 
  rm: remove regular empty file ‘test_1’? y
  [root@CentOS7 ~/dir2]#ll
  total 0
  [root@CentOS7 ~/dir2]#
    • -f 参数强制删除.

处理目录

有些命令对文件和目录都有效，而有些指对目录有效。

创建目录

  • mkdir 命令创建目录

在dir2的目录下创建了一个New_dir的目录。

[root@CentOS7 ~/dir2]#mkdir New_dir
   [root@CentOS7 ~/dir2]#ll
   total 0
   drwxr-xr-x. 2 root root 6 Jun 29 14:55 New_dir
  • 创建多个目录和子目录 - p 参数。。
[root@CentOS7 ~/dir2]#mkdir -p New_dir/one_dir/two_dir
[root@CentOS7 ~/dir2]#ls -R New_dir/
New_dir/:
one_dir

New_dir/one_dir:
two_dir

New_dir/one_dir/two_dir:

删除目录

删除目录的基本命令rmdir。

  • 目录中有文件是无法直接使用rmdir命令删除的

  • 使用-r选项向下进入目录，删除其中的文件然后删除自身 删除New_dir/目录包括里面的文件

[root@CentOS7 ~/dir2]#rm -ri New_dir/
rm: descend into directory ‘New_dir/’? y
rm: descend into directory ‘New_dir/one_dir’? y
rm: remove directory ‘New_dir/one_dir/two_dir’? y
rm: remove directory ‘New_dir/one_dir’? y
rm: remove directory ‘New_dir/’? y
[root@CentOS7 ~/dir2]#ll
total 0
  • -f同样向下删除大量目录和文件
[root@CentOS7 ~/dir2]#rm -fr dir001/ 
[root@CentOS7 ~/dir2]#ll
total 0

rm -rf命令没有警告信息，没有提示声音。威胁的工具。

查看文件内容

  • 查看文件类型： file 命令
[root@CentOS7 ~/dir2]#ll
total 4
-rw-r--r--. 1 root root 11 Jun 29 15:14 file.txt
[root@CentOS7 ~/dir2]#file file.txt 
file.txt: ASCII text

  • 查看整个文件

    • cat 命令显示文本文件中所有数据的得力工具。直接显示文件全部内容，无法控制

      • -n 参数会给所有的行，加上行号。
      [root@CentOS7 ~/dir2]#cat -n file.txt 
 1  aaa
 2
 3  bbb
 4
 5  dd
 6
 7  d
 8  c
 9
10
      • -b 只给有文本的行加上行号。
      [root@CentOS7 ~/dir2]#cat -b file.txt  
 1  aaa

 2  bbb

 3  dd

 4  d
 5  c
      • -T参数会用^T字符组合去替换文本中的所有制表符。

    • more命令，会在每页数据之后停下来

    • less命令，一次显示一屏的文本文件。能够识别上下键翻页。

  • 查看部分文件

    • tail命令，显示文件最后几行的内容。默认显示文件末尾10行。

      • tail -n 2 文件名 （显示文件最后两行）
      • tail -2 文件名
      • -f 是tail的付出特性它允许你在其他进程使用该文件时查看文件的内容。tail 命令会保持活动状态，并不断显示添加到文件中的内容。这是实时监测系统日志的绝妙方式。

    • head 命令显示默认前10行。

      • head -n 2 文件名
      • head -2 文件名

小结

  • touch 命令非常有用，可以创建空文件和变更已有文件的访问时间或修改时间。

  • 本章还介绍了如何使用 cp 命令将已有文件复制到其他位置。另外还逐步介绍了如何链接文件，给出了一种简单的方法可以实现在两个位置上拥有同一个文件且不用生成单独的副本。 ln 命令提供了这种链接功能。

  • 接着我们讲了怎样用 mv 命令重命名文件（在Linux中称为移动文件），以及如何用 rm 命令删除文件（在Linux中称为移除文件），还介绍了怎样用 mkdir 和 rmdir 命令对目录执行相同的任务。

  • 最后，本章以如何查看文件的内容作结。 cat 、 more 和 less 命令可以非常方便地查看文件 全部内容，而且 tail 和 head 命令还可查看文件中的一小部分内容。
